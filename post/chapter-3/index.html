Interactive Dashboards to Understand Data Scientist Job Locations and Job Types in 2024

March 25, 2024

Project Structure

  • Task 1: Obtained the dataset from Kaggle, ensuring we had the necessary data to analyze.

  • Task 2: Transformed simple Python scripts into web applications, making the analysis accessible through a user-friendly interface.

  • Task 3: Loaded the dataset containing job data for data scientists in 2024, preparing it for visualization.

  • Task 4: Implemented functionality to display job types in the sidebar, enhancing user navigation and interaction.

  • Task 5: Developed interactive bar plots and pie charts to visually represent key insights from the data.

  • Task 6: Created visualizations showing the number of jobs by job types, accompanied by raw data for deeper analysis.

  • Task 7: Generated word clouds for different work arrangements (onsite, hybrid, and remote), providing a visual summary of job trends in each category.

Through these tasks, I successfully built an interactive dashboard that enables users to explore and understand data scientist job trends in 2024 with ease.
