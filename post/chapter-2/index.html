HR Morning Email Automation

March 23, 2024

In this project, I developed an Email Automation Package using Python with various functionalities. Here’s a breakdown of the tasks carried out:

Task 1: Retrieving Content from the Internet

  • Automated the extraction of random quotes and saved them to a CSV file using web scraping techniques with BeautifulSoup.
  • Integrated weather forecast retrieval for a one-day period within a three-hour time frame using a REST API.
  • Implemented a random learning generator utilizing Wikipedia’s REST API.

Task 2: Formatting Email Content

Created functionalities to format email content with both plain text and HTML files.

Task 3: Email Sending Script

Developed a Python script for sending emails with Gmail authentication.

Task 4: Task Scheduling

Scheduled tasks to run at specific time periods for automation purposes.

Task 5: Admin GUI

Built an admin graphical user interface (GUI) using Tkinter, Python’s built-in GUI toolkit.

Task 6: Configuration Management

Implemented functionality to save changing configurations to a JSON file for easy management.

Task 7: Packaging

Packaged the entire project using PyInstaller for distribution and ease of installation. Overall, this project aims to streamline email automation processes by integrating various functionalities and providing a user-friendly interface for administration.
