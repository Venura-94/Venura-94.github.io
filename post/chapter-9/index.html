Road Sign Classification with Convolutional Neural Networks

April 1, 2024

Introduction

In this project, I utilized Convolutional Neural Networks (CNNs) to classify 43 different traffic sign images. The goal was to create a model with practical applications, especially in the context of self-driving cars. Below is a summary of the steps involved in the project.

1. Understanding Image Data

I began by importing necessary libraries and datasets. However, due to the large size of the dataset, it wasn’t feasible to include it in the GitHub repository.

2. Data Preprocessing

  • Visualized the images to gain insights into the dataset.
  • Converted images to grayscale and performed normalization to prepare them for model training.

3. Model Building

  • Constructed a Convolutional Neural Network using Keras with TensorFlow 2.0 backend.
  • Compiled the model and trained it on the preprocessed dataset.

4. Model Evaluation

  • Assessed the performance of the trained CNN using various key performance indicators (KPIs) such as accuracy, precision, and recall.

Learning Objectives Achieved

By completing this project, I achieved the following learning objectives:

  1. Understanding CNNs: Gained insights into the theory and intuition behind Convolutional Neural Networks.
  2. Data Handling: Learned to import, visualize, and preprocess dataset images using Python libraries.
  3. Model Development: Built a CNN model using Keras with TensorFlow backend.
  4. Model Evaluation: Evaluated the model’s performance using various KPIs.

Docker Integration

To make the model deployment-ready, I integrated it with Docker in a step-wise manner:

  1. Model Saving: Saved the trained model according to the TensorFlow serving models.
  2. Dockerization Process: Pulled image from Docker Hub and created a TensorFlow serving image.
  3. Container Execution: Ran the Docker container with TensorFlow serving image, mounted model directory to TensorFlow model in the container, and added necessary environment variables.
